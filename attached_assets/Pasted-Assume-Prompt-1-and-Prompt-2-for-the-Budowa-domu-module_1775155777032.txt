GOAL
Create a practical investor-oriented stage planner for building a single-family house in Poland.

IMPORTANT PRODUCT RULES
- This module must not act like a universal structural DIY manual.
- It must act as an investor assistant:
  - stage overview
  - sequencing guidance
  - site-readiness checks
  - role-of-professional checks
  - warning states
  - energy/quality reminders
  - documentation reminders
- For foundations, structure, roof, waterproofing, installations, and envelope works:
  - do not present product-independent execution details as absolute rules
  - add a strong note that the project documentation and selected system/product technical documentation have priority
- Keep beginner-friendly language.

Add a full "Build Stages" section inside the House Build module.

Create these main stages:
- Plot and site preparation
- Foundations and foundation waterproofing
- Structural walls / structural shell
- Roof structure and roof covering
- Windows and exterior doors
- Internal rough installations
- Insulation / airtightness / energy standard planning
- Facade and external finishing
- Internal finishing and fit-out

2. PLOT AND SITE PREPARATION STAGE

Do not turn this into a contractor field manual.

3. FOUNDATIONS STAGE
Create an investor checklist for foundations.

The app should explain in simple language:
- foundations must follow the project design and site conditions
- waterproofing and ground-moisture/water conditions matter
- this stage should not be treated as generic one-size-fits-all execution
- investor should verify:
  - excavation readiness
  - reinforcement/design compliance responsibility
  - waterproofing system selection responsibility
  - curing/waiting and inspection checkpoints
  - backfilling only after the appropriate stage checks

Add warning notes:
- foundation errors are difficult and expensive to correct later
- waterproofing choice depends on site and design conditions
- project/system documents have priority

4. STRUCTURAL WALLS / SHELL STAGE
Create an investor stage for structural walls and shell progression.

Include guidance such as:
- first structural layers and geometry accuracy are critical
- wall system rules depend on selected technology/system
- openings, lintels, ring beams, floor/slab interfaces, and structural continuity must follow the project
- do not present wall execution as a universal single recipe

Add investor checks like:
- confirm selected wall technology/system
- confirm team familiarity with the selected system
- confirm stage acceptance before moving to next structural stage
- confirm moisture/weather protection for unfinished structure where relevant

5. ROOF STAGE

Add warning notes:
- roof system details vary by product/system
- ventilation and detailing should not be guessed
- project/system documentation has priority
